== Description ==

NS Basics is a theme framework used for plugins and themes built by Nightshift Creative. It provides easy creation of admin pages and setting panels.

Full documentation here: [Docs](https://products.nightshiftcreative.co/docs/ns-basics/)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
